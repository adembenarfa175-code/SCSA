# 1-Bit Computer Instruction Set Architecture (ISA) Reference
# Instruction Format: 8 bits total [ OPCODE (4 bits) | OPERAND/ADDRESS (4 bits) ]

## 1. CONTROL AND HALT OPERATIONS

| Opcode | Mnemonic | Format | Description |
| :----: | :------- | :----- | :---------- |
| 0000   | HLT      | HLT 0000 | Halts the CPU execution. Stops the clock/program counter. |
| 0001   | NOP      | NOP 0000 | No Operation. CPU consumes one clock cycle and moves to the next instruction. |

## 2. DATA TRANSFER OPERATIONS (Loads)

| Opcode | Mnemonic | Format | Description |
| :----: | :------- | :----- | :---------- |
| 0010   | LDA      | LDA [xxxx] | Load the 1-bit value from RAM address [xxxx] into the Accumulator (A). |
| 0011   | STA      | STA [xxxx] | Store the 1-bit value from Accumulator (A) into RAM address [xxxx]. |
| 0100   | LDI      | LDI xxxx | Load Immediate. Load the 4-bit Operand directly into a specific 1-bit component (e.g., A = xxxx[0]). |

## 3. ARITHMETIC/LOGIC OPERATIONS (ALU)

| Opcode | Mnemonic | Format | Description |
| :----: | :------- | :----- | :---------- |
| 0101   | NOT      | NOT 0000 | Invert the value in the Accumulator (A = NOT A). (Uses ALU for operation). |
| 0110   | XOR      | XOR [xxxx] | Perform 1-bit XOR operation: A = A XOR RAM[xxxx]. (Logic Operation). |
| 0111   | AND      | AND [xxxx] | Perform 1-bit AND operation: A = A AND RAM[xxxx]. (Logic Operation). |

## 4. BRANCHING AND JUMP OPERATIONS

| Opcode | Mnemonic | Format | Description |
| :----: | :------- | :----- | :---------- |
| 1000   | JMP      | JMP [xxxx] | Unconditional Jump. Set the Program Counter (PC) to the address [xxxx]. |
| 1001   | JNZ      | JNZ [xxxx] | Jump if Not Zero (Jump if A=1). Set PC to [xxxx] ONLY if Accumulator (A) is 1. |

## 5. INPUT/OUTPUT OPERATIONS

| Opcode | Mnemonic | Format | Description |
| :----: | :------- | :----- | :---------- |
| 1010   | IN       | IN 0000  | Read the state of the designated Input Switch into the Accumulator (A). |
| 1011   | OUT      | OUT 0000 | Write the value of the Accumulator (A) to the designated Output LED. |
